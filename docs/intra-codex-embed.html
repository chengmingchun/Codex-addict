Intra Codex

内网 Agent Workbench
本地 CLI 能力池

Intra Codex 把 opencode、cac、codeagent 等本地 agent CLI 汇聚成统一桌面工作台。它负责项目、会话、Skills、上下文文件和并发调度，底层能力仍由公司内网已有 agent 执行。

项目会话 按文件夹管理上下文和历史，像 Codex 一样持续推进任务。
Agent 配置 命令、参数、并发和 prompt 传入方式都可以在界面里调整。
Skills 注入 从 Markdown 目录加载团队工作流，按任务组合使用。
上下文文件 在文件树勾选关键文件，后端安全读取并按预算打包。
选择项目 → 选择 Agent → 加载 Skills → 勾选上下文 → CLI 执行
Windows 单 exe 分发，无需在内网机器执行 npm build。 下载 intra-codex.exe
